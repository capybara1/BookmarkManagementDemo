# espial

Project resources:

- https://github.com/jonschoning/espial
- https://github.com/jonschoning/espial-docker

Features:

- Web frontend
- Complex search filters
- Tags
- Notes
- Title by fetch
- Flag as "read later"
- Favourites
- Bookmarklet
- Markdown support
- Import from json
- Export to json
- Persistent storage: sqlite3
